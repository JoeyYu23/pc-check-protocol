# 卖家使用指南

你好！如果你正在出售二手电脑，买家可能会要求你运行这个工具来验证硬件真实性。

**不用担心，这个工具非常安全，根据买家选择的测试模式，大约需要 5-40 分钟。**

---

## 这个工具是做什么的？

这是一个**整机验机工具**，帮助买家确认：
- 你的 CPU、GPU 型号是否和你描述的一致
- 硬盘是否健康，没有坏道或过度磨损
- CPU 和显卡能不能稳定运行（跑压力测试不崩溃、不过热）
- 内存是否正常

---

## 这个工具安全吗？

**安全。** 以下是明确承诺：

- **不会远程控制你的电脑** — 没有远程桌面、没有后门、没有网络连接
- **不会访问你的个人文件** — 不碰你的文档、照片、微信记录、密码
- **不会在后台运行** — 脚本跑完就结束，不安装任何常驻程序
- **不上传任何数据** — 所有结果只保存在你电脑的 `output/` 文件夹里
- **代码完全公开** — 所有 `.ps1` 文件都是普通文本，用记事本就能打开查看

如果你想自己查看代码，右键任意 `.ps1` 文件，选"用记事本打开"，里面写的都是注释，你可以看到每一步在做什么。

---

## 四种测试模式

运行后会弹出菜单让你选择：

| 模式 | 大概时间 | 适合场景 |
|------|---------|---------|
| [1] 快速验机 | 约5分钟 | 时间紧，买家只看系统信息和硬盘健康度 |
| [2] 标准验机 ★ | 约15分钟 | **一般选这个就够了**，系统信息 + GPU压力 + 硬盘 |
| [3] 完整验机 | 约30-40分钟 | 买家想要完整测试，GPU+CPU+内存+硬盘全测 |
| [4] 自定义 | 按选项而定 | 买家指定具体要测哪几项 |

**一般选 [2] 标准验机 就够了。** 如果买家没有特别要求，直接选 2 即可。

---

## 操作步骤（一步步来）

### 第一步：下载工具包

买家会给你发一个下载链接，点进去下载 ZIP 文件，解压到桌面（或任意地方）。

### 第二步：下载测试软件（买家可能已经打包好了）

这些都是免费的专业硬件测试软件，各大硬件论坛都在用：

| 软件名 | 下载地址 | 怎么放 |
|--------|---------|--------|
| HWiNFO64 | https://www.hwinfo.com/download/ | 下载 portable 版，放进 `tools/` |
| FurMark | https://geeks3d.com/furmark/ | 下载安装，安装目录选 `tools\FurMark\` |
| OCCT | https://www.ocbase.com/download | 下载安装，安装目录选 `tools\OCCT\` |

> 如果买家发来的 ZIP 里已经包含 `tools/` 文件夹，可以跳过这步。

放好后，`tools/` 文件夹看起来应该像这样：

```
tools/
  HWiNFO64.exe
  FurMark/
    FurMark.exe
  OCCT/
    OCCT.exe
```

### 第三步：运行验机

1. 找到解压出来的文件夹
2. **右键** `run_windows.bat`
3. 选择"**以管理员身份运行**"
4. 黑色命令行窗口会弹出，会出现菜单让你选测试模式
5. **一般直接输入 2，按 Enter 就行**

> 为什么需要管理员权限？有些硬件信息（比如内存频率、硬盘 SMART 数据）需要管理员权限才能完整读取。不以管理员运行也能跑，只是部分数据可能显示 N/A。

### 第四步：等待测试完成

**标准验机（选 2）** 程序会自动运行：
- **收集系统信息**（1分钟以内）：CPU、主板、内存、显卡型号、驱动版本
- **FurMark GPU 压力测试**（约5分钟）：显卡全力工作，测试稳定性。脚本会自动启动，你只需要等着
- **硬盘健康检查**（约1分钟）：自动读取硬盘 SMART 数据，不需要额外操作

如果买家要求 **完整验机（选 3）**，还会有：
- **OCCT VRAM 测试**：程序会打开 OCCT，提示你点一下"开始"按钮
- **OCCT CPU 测试**：同样，提示你在 OCCT 里切换到 CPU 测试并点开始
- **OCCT 内存测试**：同上

**遇到 OCCT 步骤时，脚本会用中文告诉你操作步骤，按提示做就行。**

期间电脑风扇会变响、温度会升高，这是正常的，不用担心。

### 第五步：发送结果

测试完成后，命令行窗口会提示结果文件位置，类似：

```
结果目录: output\20240315_143022\
压缩包:   output\pc_check_20240315_143022.zip
```

把这个 **ZIP 文件**发给买家就行了。

## 各工具说明

- **HWiNFO64**：专业硬件监控，每秒记录温度、风扇转速、功耗。测试完后买家可以看到温度曲线是否正常
- **FurMark**：俗称"甜甜圈"，是显卡圈最知名的烤机工具，让显卡全力工作，看会不会崩溃或过热
- **OCCT**：可以测试显存颗粒（VRAM）、CPU 稳定性、内存稳定性

## 常见疑虑解答

**"我担心这个程序是恶意软件"**

完全理解！建议你用 VirusTotal（https://www.virustotal.com）扫描 `run_windows.bat` 和 `scripts/` 文件夹里的 `.ps1` 文件。这些都是纯文本脚本，没有任何可执行二进制代码。

**"运行完要把这个软件卸载吗？"**

不需要卸载，直接删除整个文件夹即可（包括 `tools/` 里的工具）。没有安装任何东西到系统目录。

**"测试过程中温度很高正常吗？"**

烤机时温度升高是正常现象。不同硬件的正常温度范围不同：

| 硬件 | 压力测试正常范围 | 注意值 |
|------|--------------|--------|
| GPU 核心温度 | 70-90°C | 95°C+ 需注意 |
| CPU 温度 | 70-90°C（视散热器） | 100°C+ 需注意 |
| 硬盘温度 | 30-50°C | 55°C+ 需注意 |
| 内存温度 | 40-60°C | 70°C+ 需注意 |

只要风扇正常运转、测试没有中断，温度在以上范围内就没有问题。

**"买家能看到我电脑里的什么信息？"**

只有硬件信息：CPU 型号、主板型号、内存容量、显卡型号和驱动版本、磁盘容量和序列号。你的文件、账号、浏览器记录等完全不在采集范围内。

## 如果出现问题

遇到任何问题，把命令行窗口截图发给买家说明，或者查看 `output/` 目录里的 `session_transcript.log` 文件（用记事本打开），里面有完整的运行记录。
